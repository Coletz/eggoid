### What is this repository for? ### 

This repository is for Eggon's Android library, Eggoid. 

### Installation ### 

To import the library put in your app's module: 
repositories { 
    maven { 
        url 'todo' 
        credentials { 
            username 'username' 
            password 'password' 
        } 
    } 
} 
dependencies { 
    compile 'co.eggon:eggoid:1.0'
}
